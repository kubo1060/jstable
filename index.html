About
This is simple javascript/jquery table for showing data from a json url source. It has self-generated element functions such as thead, tbody, pagination elements, simple filter. The columns are self-generated by keys from your json api too. Its not a professional one, im still learning javascript, thatswhy im open for criticism.

How to use it
  • Create object called for example options like in example bellow.
  • Add him neccessary properties: Table name: id, Our div, where table will be generated: divId, table style: styleClass (in example i used bootstrap), pageSize
  • As we have our options objects we can call CreateTable(options) function for each one where the only parameter is options object
  • Table should have been generated and placed into our div/divs
